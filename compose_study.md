## AI 概念

### Agent (智能体)

在人工智能领域，Agent 是一个能够感知其环境、进行思考和决策，并采取行动以实现特定目标的自主实体。

在大型语言模型 (LLM) 的背景下，Agent 通常指一个系统，它以 LLM 为核心，并具备以下能力：

1.  **推理与规划**: Agent 可以利用 LLM 的强大语言理解和生成能力来分析复杂任务，将其分解为更小的、可执行的步骤。
2.  **工具使用**: Agent 不仅仅是生成文本，它还可以与外部工具或 API 进行交互。例如，它可以运行代码、执行 shell 命令、搜索网络、访问数据库等。
3.  **自主性**: 在给定一个高级目标后，Agent 可以在没有持续人类干预的情况下，自主地执行一系列操作来完成任务。

简单来说，你可以把 Agent 看作一个能够使用工具来完成复杂任务的、由语言模型驱动的 "思考者" 和 "执行者"。

### Token (令牌)

在自然语言处理 (NLP) 和大型语言模型中，Token 是模型处理文本的基本单位。当模型接收一段文本时，它首先会将文本分解成一系列的 Token。

- **Token 是什么？**: 对于英语来说，一个 Token 通常是一个单词或一个有意义的子词（例如，"unhappily" 可能会被分解为 "un", "happi", "ly"）。对于中文来说，一个 Token 可能是一个汉字，也可能是一个词语。
- **为什么重要？**:
    - **模型限制**: 每个模型都有一个 "上下文窗口" 的限制，即它一次能够处理的最大 Token 数量。例如，一个模型的上下文窗口可能是 4096 个 Token。超出这个限制的文本将无法被处理。
    - **计费**: 大多数商业 LLM 服务的计费方式是基于输入和输出的 Token 总数。使用的 Token 越多，费用就越高。

理解 Token 的概念对于有效和经济地使用大型语言模型至关重要。
