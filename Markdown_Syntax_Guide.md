# Markdown 语法指南

Markdown 是一种轻量级标记语言，让人们能够使用易读易写的纯文本格式编写文档。

## 标题

使用 `#` 号可表示 1-6 级标题，一级标题对应一个 `#` 号，二级标题对应两个 `#` 号，以此类推。

```markdown
# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题
```

## 字体

- **粗体**: 使用两个星号 `**` 或下划线 `__` 包裹。
  `**这是粗体**` 或 `__这是粗体__`

- *斜体*: 使用一个星号 `*` 或下划线 `_` 包裹。
  `*这是斜体*` 或 `_这是斜体_`

- ***粗斜体***: 使用三个星号 `***` 或下划线 `___` 包裹。
  `***这是粗斜体***` 或 `___这是粗斜体___`

- ~~删除线~~: 使用两个波浪线 `~~` 包裹。
  `~~这是删除线~~`

## 列表

### 无序列表

使用星号 `*`、加号 `+` 或是减号 `-` 作为列表标记：

```markdown
* 项目1
* 项目2
  * 子项目 A
  * 子项目 B
```

### 有序列表

使用数字并加上 `.` 号来表示：

```markdown
1. 第一项
2. 第二项
3. 第三项
```

## 引用

使用 `>` 符号来创建块引用。

```markdown
> 这是一个引用。
>
> > 这是一个嵌套的引用。
```

## 代码

###行内代码

使用反引号 `` ` `` 包裹。

`这是一个行内代码`

### 代码块

使用三个反引号 ``` ``` 包裹代码块，并可以指定语言。

````
```kotlin
fun main() {
    println("Hello, Markdown!")
}
```
````

## 链接

格式为 `[链接名称](链接地址)`。

```markdown
[访问 GitHub](https://github.com)
```

## 图片

格式为 `![替代文本](图片地址)`。

```markdown
![GitHub Logo](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)
```

## 表格

使用 `|` 来分隔不同的单元格，使用 `-` 来分隔表头和主体。

```markdown
| 表头1 | 表头2 |
| ----- | ----- |
| 单元格1 | 单元格2 |
| 单元格3 | 单元格4 |
```

对齐方式：

- `:-`  左对齐
- `:-:` 居中对齐
- `-:`  右对齐

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :--- | :----: | ---: |
| 文本 |  文本  |   文本 |
```

## 分割线

你可以在一行中用三个以上的星号、减号、底线来建立一个分隔线。

```markdown
***
---
___
```
